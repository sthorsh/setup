  Enkel kommando
    cmd [opts] [args]
    Funksjon, builtin, extcmd, script

  Pipeline
    [!] cmd₁ [| or |& cmd₂] ...

  Liste
    pipeline₁ [; or & or && or || pipeline₂] ... [; or & or \n]

  (())
    (( arithmetic-expr ))
  
  [[]]
    [[ bool-expr ]]

if
  if commands; then commands; [elif commands; then commands;] ... [else commands;] fi
  
  case
    case word in [pattern [| pattern]...) cmds ;;]... esac

  select
    select name [in words ... ;] do cmds; done
  
for
  for name [in words ... ] ; do commands; done
  Name binder seg til positional params hvis words mangler
  Words støtter filename expansion

for
  for (( expr₁ ; expr₂ ; expr₃  )) ; do cmds ; done
   
  while
    while cmds₁; do cmds₁; done
  
  until
    until cmds₁; do cmds₂; done
  
  {}
    { list }
    Brukes for gruppering
    Liste må ha terminator ; or & or \n
    Kjører i current shell context
  
  ()
    ( list )
    Brukes for gruppering
    Oppretter subshell execution environment
    Kjører i subskallet
  
  Coprosess
    coproc [name] command [redir]

----------------------
In general
  A semicolon may be replaced with a newline
  All loops may use break [N] or continue [N]
