API LABELS

Label
  Key/value-par
  Assosisert med objekter/ressurser

Label selector
  Identifiserer en gruppe (mengde) objekter
  The core grouping primitive in kubernetes

Labels CRUD
  $ kubectl label pod <name> key=value (create)
  $ kubectl get pod <name> --show-labels (read)
  $ kubectl get pod <name> --label-columns key₁,... (read)
  $ kubectl label pod <name> key=value --overwrite (update)
  $ kubectl label pod <name> key- (delete)

Selector CLI
  $ kubectl <cmd> <resource> --selector <query>

Selectors API
  deployment.spec.selector: app=nginx
  replicaset.spec.selector: app=nginx, pod-template-hash=1609499587
  service.spec.selector: app=nginx
  pod.spec.nodeSelector → node (pod scheduling)

Query
  key
  !key
  key=value
  key!=value
  key in (str, ...) (or)
  key notin (str, ...)
  query₁, query₂, ... (and)
