KOMMANDOER

Kommandoer
  [name=value]... cmd [args] [redir]           enkeltkommando

  [!] command [| command]...                   pipeline

  pipeline [; pipeline]...  [; or & or \n]     sekvensiell liste
  pipeline [& pipeline]...  [; or & or \n]     asynkron liste
  pipeline [&& pipeline]... [; or & or \n]     og-liste
  pipeline [|| pipeline]... [; or & or \n]     eller-liste

  ((expr))                                     help "(("
  [[ expr ]]                                   help [[

  if                                           help if
  case                                         help case
  select                                       help select

  for                                          help for
  while                                        help while
  until                                        help until

  ( list )                                     gruppe
  { list; }                                    gruppe

  coproc [name] cmd [redir]                    coprocess

  name() compound-command
