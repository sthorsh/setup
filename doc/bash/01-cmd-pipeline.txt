Command-/pipeline
  [env] alias|keyword|fn|builtin|file [opts] [args] [redirect] [&] [| ...]

Prosessering
  History expansion
  Splitter cmdline i cmds
  Setter opp I/O for cmdline
  For hver cmd do...
    Parse og tokeniserer vha ␣ ␉ ␤ ; ( ) < > | &
    Alias expansion
    Opening keyword
    Brace expansion {}
    Tilde expansion ~
    Param expansion $ (svak quoting)
    Cmd substitusjon $() (svak quoting)
    Aritmetisk subsitusjon $(()) (svak quoting)
    Parse og tokeniserer vha $IFS
    Pathname expansion * ?
    Command lookup
    Setter opp I/O for cmd mm
    Utfører cmd
  Done

Quoting
  Uten
  Svak ""
  Sterk ''

Brace expansion
  pre{x..y..N}post
  pre{str,...}post
  pre{...}{...}post

Tilde expansion
  ~  = $HOME
  ~+ = $PWD
  ~- = $OLDPWD

Command lookup
  Function
  Builtin
  Executable (-x) fil i $PATH
