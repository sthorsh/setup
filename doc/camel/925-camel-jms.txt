Header mapping
- Hvis navn inneholder .: erstatter med _DOT_
- Hvis navn inneholder -: erstatter med _HYPHEN_ 
- Hvis navn starter med JMS: dropper header
- Verdi: BigInteger, BigDecimal, CharSequence, Date → String
- Hvis verdiens type ikke er primitiv/wrapper: dropper header

Body type mapping
- byte[] mfl → BytesMessage → byte[]
- Map → MapMessage → Map
- Serializable → ObjectMessage → Object
- String → TextMessage → String
- ? → StreamMessage → no mapping
