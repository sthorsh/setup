CLI

$ docker
  $ docker<CR>                      opts, subcmds, cmds
  $ docker subcmd<CR>               usage, cmds
  $ docker subcmd cmd --help        usage, opts
  $ docker version                  klient, server

$ service
  $ service docker status           service status
  $ service docker start            start server
  $ service docker stop             stop server
  $ service docker restart          restart server

$ journalctl
  $ journalctl -u docker.service
