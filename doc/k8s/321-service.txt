SERVICE               

API
  apiVersion            v1
  kind                  Service
  metadata              ObjectMeta
    name                my-svc (k8s dns entry)
    namespace           default
    labels              my-key=my-val
    ...
  spec                  ServiceSpec
    type                String
    selector            object
    ports               ServicePort
      nodePort          30001-32767 e.g http://192.168.99.100:30001/
      port              8080
      protocol          TCP or UDP
      targetPort        8080
  status                ServiceStatus (ro)

Service
  ETT entrypunkt        for N pods
  Et FAST entrypunkt    dns|ip:port
  Ekstern eksponering   ut av noden
  Lastbalansering       random over endpoints

Service type
  ClusterIP             pod → svc → pod
  NodePort              ext → np_svc → pod
  LoadBalancer          ext → lb_svc → np_svc
  ExternalName          pod → en_svc → ext_svc

Service selector
  A/B deployment        e.g 1.1 → 1.2
  Canary deployment     e.g 1.1 → 1.2
