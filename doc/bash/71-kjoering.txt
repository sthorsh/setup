Miljøet (kun bash ?)
  Et skall har et miljø (env)
  Et miljø er en liste med name=value params/funksjoner
  Et skall eksporterer sitt miljø til sine barn (export) 
  Env-=navn: declare +x name
  Env-=navn: export -n name
  Env-=navn: unset name
  Env+=navn: declare -x name[=value]
  Env+=navn: export name
  Env+=navn: name=value cmd

Kjøremiljø (alle prosesser ?)
  Aliaser           alias               alias='...'
  Arbeidskatalog    pwd                 parent OR cd ...
  Fildeskriptorer   lsof -p $$          parent OR exec N...
  Funksjoner        declare -F          parent env OR name(){...}
  Opsjoner (set)    $-; or set -o       bash -... OR set -o ...
  Opsjoner (shopt)  shopt -p            shopt -s ...
  Params            declare -p          parent env OR set -- ... OR name=value
  ProsessID'er      $$; $PPID; jobs -p  cmd &
  Traps             trap -p             trap ...
  Umaske            umask               parent | umask ...
