Git er VCS
  Distribuert VCS
  VCS
  Content tracker
  Persistent map: ~/,git/objects

Git er filsystem
  Fil: blob
  Katalog: tree
  Versjon: commit

Objektmodell
  Commit → commit, tree
  Tree → tree, blob
  Blob 
  Tag → commit

Branch
  En peker/ref til en commit

HEAD
  En peker/ref til current branch
  Eller en peker/ref til en commit (detached HEAD)

Tag 
  En peker/ref til en commit
  En tag er en branch som aldri flytter seg

Merge
  Fra source branch inn i target branch
  En merge er en commit med to parent commits (merge commit)
  Fast-forward merge: move current branch tip to target branch tip
  
Rebasing
  Alternativ til merge
  Gjenta commits on top of another base tip
  
Distribusjon
  Maskin1: repo (remote: maskin2)
  Maskin2: klone av repo (remote: maskin1)
  Ved kloning: git oppretter remote origin
  Et prosjekt kan ha ≥1 remotes: $ git remote add <new_remote> <url>
  Hver remote kan ha ≥1 urls: $ git remote set-url --add <remote> <new_url>
