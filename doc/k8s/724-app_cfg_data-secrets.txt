  Sensitive data

Aksess fra app
  Alt1: env vars
  Alt2: volumes dvs fs

Definisjon
  $ kubectl create cm app-cfg --from-literal key₁=val₁ ...

Bruk
  pod.spec.containers.env
  pod.spec.containers.envFrom (✓)
  pod.spec.containers.volumeMount (✓)

------------
SECRETS  
E.g: API tokens, passord, keys, sertifikater
Secrets er namespaced
Secrets er lagret i etcd
Secrets er base64 enkodet
Secrets kan være kryptert
Secrets kan leses/refereres vha env_vars
Secrets kan leses/refereres vha filer (volumes)
Secrets kan kun leses av pods i samme namespace
Utilgjengelige secrets vil hindre en pod fra å starte
