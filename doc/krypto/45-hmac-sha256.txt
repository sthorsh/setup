SYMMETRISK KRYPTOGRAFI
  
HMAC-SHA256
  M opprinnelig melding
  K symmetrisk hemmelig nøkkel
  ipad inner padding
  opad outer padding
  H hashfunksjon dvs sha256
  b block size dvs 512 bits
  n hash value size dvs 256 bits
  
  Pad K til b bits
  Beregn padded_K XOR ipad -> S₁
  Prepend S₁ til M
  Beregn hash S₁ + M dvs H(IV, S₁||M) -> hashverdi₁
  Pad n-bits hashverdi til b bits (H₁)
  Beregn padded_key XOR opad -> blokk S_outer
  Prepend S_outer til H₁
  Hash S_outer + H₁ dvs H(IV, S_outer || H₁) -> HMAC(M, K)
