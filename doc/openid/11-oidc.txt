OIDC

Parter
  Sluttbruker: end user EU
  Brukeragent: user agent UA
  Klientapp: relying party RP
  Autentiseringsserver: OpenID provider OP
  API provider: resource server RS

RP
  Serverapp
  JS e.g SPA
  Førstepartsapp
  API provider

Flyter
  Authz code
  Implisitt: client secret n/a
  ROPC
  Client credentials: EU n/a
  OpenID hybrid

RP → flyt
  Serverapp: authz code
  JS e.g SPA: implisitt
  Førstpartsapp: ROPC
  API: client credentials

Autentisering av EU
  Brukerakkreditiver i OP (authz code, implisitt)
  Brukerakkreditiver i RP (ROPC)

Autentisering av RP
  Cf RFC 7617
  Cf RFC 6749
  POST HTTP body m/klientakkreditiver

  /token: utenom UA, TLS required
  /userinfo: utenom UA, TLS required
  /introspect: token introspection
