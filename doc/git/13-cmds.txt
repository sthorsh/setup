init
  Lag .git; lag HEAD → ?

status
  Branch
  No commits yet
  Untracked files
  Changes not staged for commit
  Changes to be committed
  Nothing to commit, working tree clean

--------------------------------------------------
add
  Add til index

mv
  Move in wtree; move in index

rm  
  Remove from wtree; remove from index

commit
  Lag commit; flytt current branch (; flytt HEAD)

revert
  Revert commmit

----------------------
branch
  F1: list [--all|--remotes] branches
  F2: lag ny branch fra HEAD [el <start-point>]

switch
  Flytt HEAD; oppdater index; oppdater wtree

merge
   Fast-forward merge: ingen ny commit
   Rekursiv merge: _en_ ny merge commit

rebase
  Rebase current branch oppå en annen branch

----------------------
restore
  Discard fra wtree: git restore <file>
  Unstage fra index: git restore --staged <file>

reset
  F1: reset index fra repo for <path>
  F2: reset HEAD (soft); reset index (mixed); reset wtree (hard)
