Komponenter
  Kubelet
  Kube proxy
  Container runtime
  
Kubelet
  En agent
    Poller API-server for workloads
    Kjører pods
    Rapporterer feil til API-server
    Eksponerer <ip>:10255/pods
    Eksponerer <ip>:10255/health
    Eksponerer <ip>:10255/spec
    ...            
    
  Kube proxy
    En nettverksproxy
    Enabler networking
    Gir hver pod en ip-addr
    Gir hver container i pod en tcp-port
    Sørger for lastabalansering for pods → service → pods
    ...
  
  Container runtime
    Kjører containere
    Cf docker runtime
