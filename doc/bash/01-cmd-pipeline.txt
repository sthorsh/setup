Command-/pipeline
  [env] alias|keyword|fn|builtin|file [opts] [args] [redirect] [&] [| ...]

Prosessering
  History expansion
  Splitter cmdline i cmds
  Setter opp I/O for cmdline
  For hver cmd do...
    Parse og tokeniserer vha ␣ ␉ ␤ ; ( ) < > | &
    Alias expansion
    Opening keyword
    Brace expansion {}
    Tilde expansion ~
    Param expansion $ 
    Cmd substitusjon $()
    Aritmetisk subsitusjon $(())
    Parse og tokeniserer vha $IFS
    Pathname expansion * ?
    Command lookup
    Setter opp I/O for cmd mm
    Utfører cmd
  Done

Brace expansion
  pre{x..y..N}post
  pre{str,...}post
  pre{...}{...}post

Tilde expansion
  ~  = $HOME
  ~+ = $PWD
  ~- = $OLDPWD

Command lookup
  Function
  Builtin
  Executable (-x) fil i $PATH

Quoting
  '': starter med command lookup
  "": param exp, cmd- og aritmetisk sub
  Uten quoting: alle trinn gjennomføres
