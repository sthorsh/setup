OAUTH2

OAuth2
  Et rammeverk for autorisasjon
  En protokoll for delegering
  Laget spesielt for HTTP API'er

Alternativer
  Credential sharing: oppgir brukernavn/passord til app
  Cookies: oppgir brukernavn/passord til API server (utsatt for csfr)
  API nøkler: kan ikke lagres trygt i feks SPA
  OAuth2: ✓

Use case  
  En måte for Internet-brukere å gi websites eller applikasjoner aksess til 
  sin informasjon på andre websites uten å gi dem passordet på disse sitene

Use case  
  En måte for Facebook, Google, Microsoft, Twitter mfl å tillate en bruker
  å dele informasjon om sin konto med tredjeparts websites og applikasjoner

Parter
  Ressurseier: bruker (RO)
  Brukeragent (UA)
  Autorisasjonsserver (AzS)
  Klient: applikasjon (app)
  Beskyttet ressurs: HTTP API (api)

Flyter
  Code flow
  Implicit flow
  Resource owner credentials flow: trusted client
  Client credentials flow: service-to-service (m2m)

okens
  Aksesstoken: sikker utveksling forutsetter TLS
  Refreshtoken: optional
