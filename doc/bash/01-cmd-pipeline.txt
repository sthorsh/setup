Cmd-/pipeline
  [env] fn|builtin|cmd|script [opts] [args] [redirect] [&] [| ...]

Prosessering
□ Split cmdline i cmds
□ Sett opp cmdline I/O
□ Splitt cmd i tokens
□ Hvis 1st token er keyword: les neste cmd & loopback
  Alias expansion
  Brace expansion
  Tilde expansion
  Param expansion
□ Command substitution
□ Arithmetic substitution
□ Word splitting
□ Asterisk/pathname/wildcard expansion
□ Command lookup: function|builtin|cmd|script
□ Sett opp cmd I/O
□ Utfør cmd og fortsett med neste cmd

Brace expansion
  pre{x..y..N}post
  pre{str,...}post
  pre{...}{...}post

Tilde expansion
  ~ →  $HOME
  ~+ → $PWD
  ~- → $OLDPWD
