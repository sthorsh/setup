KOMMANDO → KJØREMILJØ

Enkeltkommandoer
  $ func                      skall
  $ builtin                   skall
  $ ./filename                subprosess (direkte)
  $ filename                  subprosess (pathsøk)
  $ bash filename             subprosess (. → pathsøk, hash empty)
  $ bash filename.sh          subprosess (. → pathsøk, hash empty)

Pipelines
  $ func OR builtin | ...     subskall

Lister
  $ pipeline & ...            subskall

Sammensatte kommandoer
  $ ( list )                  subskall

Ekspansjoner
  $ $( expr )                 subskall (ctrl-alt-e)
  $ >( expr )                 subskall (ctrl-alt-e)
