Registrering
  App registreres i autoriasjonsserver

Token request
  Server → POST https://authzserver/token?
    grant_type=client_credentials&
    scope=resource& (api/svc)
    Authorization: Basic (client_id:secret)

Token response
  {
    access_token: header.claims.signature, (jwt)
    expires_in: 3600,
    token_type: Bearer,
    refresh_token: xyz
  }

Ressursaksess
  App → GET https://resourceserver/api|svc|resource
    Authorization: Bearer access_token (rfc 6750)
