Syntaks
  declare -x              vise  
  export                  vise 
  env                     vise 

  declare -x name=value   deklerare og eksportere
  export name=value       deklarere og eksportere

  declare -x name         eksportere
  export name             eksportere var
  export -f name          eksportere funksjon

  name=value cmd          eksportere midlertidig
  
  declare +x name         fjerne
  export -n name          fjerne

  unset name              slette
  
params og funksjoner
ekposrterer til barn, ALLE kommandoer "arver"

Miljø
  Et liste av navn=verdi,...
  Bash har et miljø, som det eksporter til sine barneprosesser
  Dvs funksjoner/builtins og script/os-cms hhv bruker og arver bash' miljø
  
Kjøremiljø (alle prosesser ?)
  Aliaser           alias               alias='...'

  Fildeskriptorer   lsof -p $$          parent OR exec N...
  Funksjoner        declare -F          parent env OR name(){...}
  Opsjoner (set)    $-; or set -o       bash -... OR set -o ...
  Opsjoner (shopt)  shopt -p            shopt -s ...
  Params            declare -p          parent env OR set -- ... OR name=value
  ProsessID'er      $$; $PPID; jobs -p  cmd &
  Traps             trap -p             trap ...
