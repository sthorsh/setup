SCHEDULER

Hva det er
  En master-komponent
  Default kube-scheduler

Hva den gjør
  Schedulerer workloads i clusteret
  Dvs binder nye pods til noder i clusteret

Hva den tar hensyn til
  Ressurser (hw)
  Policy constraints (sw)

Schedulering
  Scheduler watches API server for unscheduled pods
  Scheduler velger node for ny pod (node selection)
    Filtrering
      Vurderer alle noder med status ready
      Filtere: resource requests, node selectors
      Produserer en liste med noder (filtered nodes)
    Scoring
      Vurderer alle filtered nodes
      Scoring functions: policies, affinitet, taints, tolerations
      Produserer en liste med noder (feasible nodes)
    Binding
      Vurderer alle feasible nodes
      Velger node etter score (uavgjort → random)
      Oppdaterer pod.spec.nodeName (signaliserer ny workload)
  Nodes' kubelets watch API server for nye workloads
  Nodes' kubelets signalerer ctr runtime to start ctr
