  manifest.json
  Lag 1..N
  config.json (default config)

Image layers (FS)
  Lag 2: nginx, nginx.conf (RO)
  Lag 1 OS: bash, libc.so.6 (RO)

  Public: Docker Hub
  Privat: Docker Trusted Registry

  User: user/repo e.g sthorsh/web
  Public
  Privat
