TOKENISERING

Tokenisering
  Hvis EOL: avgrense token
  Hvis del-av-op: legg til op
  Hvis ugyldig del-av-op: avgrense op; start ny token
  Hvis \ ' ": les qoute
  Hvis $: les makro token
  Hvis start-av-op: avgrense token; start ny op token
  Hvis blank: avgrense token; forkast blank
  Hvis del-av-ord: legg til ord
  Hvis #: les og forkast resten av linjen
  Start nytt ord

Metategn
  space tab | ; & ( ) < > newline 

Token
  En sekvens med tegn
  Et ord eller en operator
  Inkluderer evt quotes feks foo\|bar

Ord
  En tegnsekvens uten unqoted metategn
  Et nøkkelord, et navn, ...
    
Operatorer  
  En tegnsekvens med ett el flere sammenhengende unquoted metategn

Nøkkelord
  ! 
  time
  if then else elif fi case esac select
  for in while until do done
  [[ ]]
  { } 
  coproc
  function

Navn
  Starter altid med bokstav el underscore 
  Består av bokstaver, tall el underscore
  E.g [a-zA-Z][_]{1}[a-zA-Z][0-9[_] (ish)

Kontrolloperatorer
    | |& ; & && || (  ) ;; ;& ;;& newline

Omdirigeringsoperator
    < > | &
