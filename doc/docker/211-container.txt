ARKITEKTUR

Arkitektur/modell
  N containere (VM: N OS)
  En container engine (VM: N VMs)
  Ett OS (VM: en hypervisor)
  En fysisk maskin: cpu, ram, disk, i/o

Container (NP)
  Et virtuelt OS
  En _isolert_ del at _ett_ OS (namespace)
  Med _resource_usage_limits_ applied (control group)

Container (WH)
  En container er en (eller flere) prosess(er)
  En container er en (eller flere) prosess(er) + isolasjon

Container fs
  CoW layer
  Unified "file system": aufs, overlay2, ...
  Underlying file system: ext4, ...

Unified filesystem
  Allows a stack of directories to be merged into a single mountpoint
  (cf https://integratedcode.us/2016/08/30/storage-drivers-in-docker-a-deep-dive/)
