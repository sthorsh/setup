NODE

Komponenter
  Kubelet
  Kube proxy
  Container runtime
  
Kubelet
  En agent
  Monitorerer API-server (poll, watch)
  Kjører pods (pod lifecycle)
  Rapporterer pod/node state
  Eksponerer <ip>:10255/pods
  Eksponerer <ip>:10255/health (pod live-/readiness prober)
  Eksponerer <ip>:10255/spec
  ...            
  
Kube proxy
  En nettverksproxy (iptables)
  Monitorerer API-server (poll, watch)
  Implementerer nettverking (ruting, services, lastbalanse)
  Hver pod: en ip-addresse
  Hver container: en tcp-port
  ...

Container runtime
  Kjører containere
  Cf docker runtime

TCP
  10250 kubelet
  30001-32767 node_port
