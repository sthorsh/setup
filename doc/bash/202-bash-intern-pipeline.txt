BASH INTERN PIPELINE

Inputprosessor
  Leser tegn fra terminal, fil eller streng
  Fra terminal: vha readline read-dispatch-execute-redisplay loop
  Output: linje med (strøm av, buffer med) tegn

Lexer
  Input: linje med tegn
  Deler opp linje ved metategn (tokenisering)
  Klassifiserer tokens (klassifisering)
  Beskytter tegn og ord med \, '' og "" (escaping, quoting)
  Ekspanderer history events 
  Ekspanderer alias 
  Output: strøm av klassifiserte tokens

Parser
  Input: strøm av tokens
  Setter sammen tokens til kommandoer, omdirigeringer etc
  Representerer gyldige kommandoer som command objects
  Representerer gyldige omdirigeringer som redirection objects
  Output: command objects, redirection objects, lister med ord

Makroprosessor
  Input: lister med ord
  Ekspanderer braces
  Ekspanderer tilde
  Ekspanderer params
  Kommandosubstitusjon
  Ekspanderer aritmetiske expr
  Prosessubstitusjon
  Orddeling
  Ekspanderer filnavn
  Fjerner escapetegn og quotes i ord
  Output: lister med ekspanderte ord

Execution engine
  Input: command objects, redirection objects, lister med ekspanderte ord
  Enable omdirigering
  Utføre (enkelt)kommando(er)
  Utføre jobbkontroll (signaler)
  Ved completion: disable omdirigering
  Ved completion: eksponere exit/return status 0..255
