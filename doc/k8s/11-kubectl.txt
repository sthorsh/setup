kubectl

Help
  $ kubectl ctrl-i
  $ kubectl ctrl-j
  $ kubectl options

Cmd help
  $ kubectl ARG ctrl-i
  $ kubectl ARG ctrl-j
  $ kubetcl ARG --help

Syntaks
  $ kubectl CMD TYPE NAME FLAGS
  $ kubectl CRUD_OP RESOURCE(S) OUTPUT

Tab completion
  $ kubectl g[tab] po[tab] p[tab] --[tab]

Manifester
  $ kubectl create TYPE NAME --dry-run=client -o yaml > NAME.yaml
