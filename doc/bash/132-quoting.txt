QUOTING

Quotes
  \               tegn
  ''              tegn eller ord; unntak '
  ""              tegn eller ord; unntak \ " $

Spesialtegn
  quotes          må quote (hvis tegn skal representere seg selv)
  metategn        må quote 
  #               må evt quote (kontekstavhengig)
  =               må evt quote 
  {               må evt quote
  ~               må evt quote
  $               må quote
  *               må evt quote
  ?               må evt quote
  [               må evt quote
  %               må evt quote

OS-kommandoer
  find            find . -name PATTERN
  grep            grep PATTERN file ...
  ...
