

Enkeltkommando 
   [name=value ...] [words] [redirections]
  
Pipeline
   [!] command₁ [| or |& command₂ ...]
  
Liste
   pipeline₁ [; or & or && or || pipeline₂ ...] [\n or ; or &]
  
if
  if COMMANDS; then COMMANDS; [ elif COMMANDS; then COMMANDS;  ]... [ else COMMANDS;  ] fi

case
  case WORD in [PATTERN [| PATTERN]...) COMMANDS ;;]... esac

select
  select NAME [in WORDS ... ;] do COMMANDS; done

for
  for NAME [in WORDS ... ] ; do COMMANDS; done
  for (( exp₁; exp₂; exp₃  )); do COMMANDS; done

while
  while COMMANDS; do COMMANDS; done

until
  until COMMANDS; do COMMANDS; done

Evaluer aritmetisk uttrykk (ekvivalent med let)
  (( expr ))

Utføre betingelseskommando
  [[ expr ]]

Blokk/gruppe
  { list ; }

Blokk/gruppe i subskall
  ( list )

Coprosess
  coproc [NAME] command [redirections]
