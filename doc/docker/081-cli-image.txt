  build <path|url>                  docker: sender build_context til dockerd

  save <image> -o img.tar           save til tar

  import cnt.tar                    import fra tar

  rm <image>                        remove image
  prune                             remote dangling images

  pull <register>/<repo>:<tag>      pull manifest
                                    pull layers (distribution hash)
                                    uncompress layers (content hash)
                                    store layers

  push <register>/<repo>:<tag>      push manifest

                                    push layers (distribution hash)
