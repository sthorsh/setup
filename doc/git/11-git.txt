GIT CMDs

For å forstå <cmd>
  Hvordan flytter <cmd> data og endrer <cmd> repo?
  Hvordan endrer <cmd> repo og flytter <cmd> data?

git
  $ git add <file>                cp data tree → index
  $ git am
  $ git annotate
  $ git apply
  $ git archive
  $ git bisect
  $ git blame
  $ git branch
  $ git bundle
  $ git checkout <branch>         mv HEAD in repo; cp data tree ← index ← repo
  $ git cherry
  $ git cherry-pick
  $ git clean
  $ git clone                     cp data tree ← index ← repo ← remote_repo
  $ git commit                    cp data index → repo; update repo objects and refs
  $ git config
  $ git describe
  $ git diff [--cached]           diff(tree,index) or diff(index,repo)
  $ git difftool
  $ git fetch <remote>            cp data repo ← remote_repo
  $ git filter-branch
  $ git format-patch
  $ git fsck
  $ git gc
  $ git get-tar-commit-id
  $ git grep
  $ git help
  $ git imap-send
  $ git init                      init repo
  $ git instaweb
  $ git interpret-trailers
  $ git log [-N]
  $ git merge <source> <target>   merge from source branch to target branch
  $ git mergetool
  $ git mv <file₁> <file₂>        mv file in tree; update index with new name
  $ git name-rev
  $ git notes
  $ git pull <remote>             fetch; merge
  $ git push <remote> <branch>    cp data repo → remote_repo
  $ git rebase
  $ git reflog
  $ git remote
  $ git repack
  $ git replace
  $ git request-pull
  $ git reset                     mv current branch; soft|mixed|hard reset
  $ git revert
  $ git rm <file>                 rm file from tree and index
  $ git shortlog
  $ git show
  $ git show-branch
  $ git stage
  $ git stash
  $ git status                    clean, untracked, unstaged|staged changes
  $ git submodule
  $ git subtree
  $ git tag
  $ git verify-commit
  $ git whatchanged
  $ git workspacetree
