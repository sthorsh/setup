SED

Syntaks
  sed cmd [files]
  sed opts cmd [files]
  sed opts [addr]cmd [files]
  sed opts 'cmd; cmd' [files]
  
Options                                           
  -e 'cmd' ...            --expression=script ...
  -f script-file          --file=script-file
  -i                      --in-place
  -n                      --quiet (no autoprint)
  -r                      --regexp-extended
  
Address
  N                       line N
  $                       last line
  /regex/                 lines with match 
  /regex/!                lines with no match
  addr₁,addr₂             from addr₁ to addr₂
  addr,+N                 from addr to addr + N
  
Commands
  a text                  append after line
  c text                  change/replace line
  d                       delete line
  i text                  insert before line
  s/regex/subex/igNpwem   substitute part of line
  
Subex
  &                       pattern space match
  \N                      group 0..9
  \n                      newline 
  \l                      lowercase next char 
  \u                      uppercase next char
  \L                      lowercase rest of subex 
  \U                      uppercase rest of subex
  \E                      end of \L or \U
