Enkel kommando (cmd)
  [env] name [opts] [args] [redir]
  Funksjon, builtin, script, os-cmd 
  Terminator er generelt \n eller ;
  Script/os-cmd kjører i subprosess
  Return status er cmds exit status 
  
Pipeline
    [!] command₁ [| or |& command₂] ...
    Rediriger command₁ stdout til command₂ stdin    

  Liste
    pipeline₁ [; or & or && or || pipeline₂] ... [\n or ; or &]
    Kjør pipelines sekvensielt, asynkront eller en-eller-begge
    Terminator ; kan generelt erstattes med \n

  if
    if commands; then commands; [elif commands; then commands;] ... [else commands;] fi

  case
    case word in [pattern [| pattern]...) commands ;;]... esac

  select
    select name [in words ... ;] do commands; done

  for
    for name [in words ... ] ; do commands; done
    Name binder seg til positional params hvis words mangler
    Words støtter filename expansion

  for
    for (( expr₁ ; expr₂ ; expr₃ )) ; do commands; done
    Evt med break el continue
   
  while
    while commands; do commands; done
    Evt med break el continue

  until
    until commands; do commands; done
    Evt med break el continue

  Evaluere uttrykk
    (( expr ))

  Evaluere uttryk
    [[ expr ]]

  Gruppe
    { list }
    Må terminere med ; or & or \n
    Kjører i current shell context

  Gruppe
    ( list )
    Oppretter og kjører i subshell execution environment

  Coprosess
    coproc [name] command [redir]
