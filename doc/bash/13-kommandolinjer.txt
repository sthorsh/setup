Kommando
  Sekvens av ord
  Adskilt av blank
  Terminert: kontrolloperator
  Exit status: kommandoens exit status

Pipeline
  Sekvens av kommandoer (evt m/prefix time)
  Adskilt av | eller |&
  Terminert av kontrolloperator
  Evt prefixet med time
  Utfører hver kommando i eget subskall
  Exit status: siste kommandos exit status

Liste
  Sekvens av pipelines
  Adskilt av ; & && ||
  Evt terminert av nl ; eller &
  Utfører hver kommando i samme (;) el eget subskall (&)
  Exit status: siste kommandos exit status (;) eller 0 (&)

Sammensatt kommando
  if-then-elif-else-fi
  case-in-esac
  select-in-do-done
  (( expr ))
  [[ expr ]]
  for-in-do-break-continue-done
  for-(())-do-break-continue-done
  until-do-break-continue-done
  while-do-break-continue-done
  ( list )
  { list; }

Coprosess
  coproc [NAME] command [redir]
  Utføres i et eget subskall (som &)
  Exit status: commands' exit status

Funksjon
  name () compound-command [ redirections ]
  function name [()] compound-command [ redirections ]
  Utføres i current shell context (ikke i en ny prosess)
  Exit status: siste kommandos exit status eller return n

--------------------------------------------------
Kontrolloperator
  nl    newline
  |     pipeline
  |&    pipeline
  ;     liste sekvensielt
  &     liste i parallell i eget subskall
  &&    liste OG
  ||    liste ELLER
  ()    compound command (subskall/-prosess)
  ;;    case break
  ;&    case fall-through
  ;;&   case if-match fall-through

Pipeline
  !     pipeline not
