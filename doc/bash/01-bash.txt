Bash
  Et skall for GNU OS
  En interpreter for et kommandospråk for GNU OS
  
Skall
  En makroprosessor (ekspansjon) som utfører kommandoer
  En kommandointerpreter dvs UI for GNU-kommandoer
  Et programmeringsspråk (PL) dvs å kombinere kommandoer

Bruk av skall
  Interaktivt: input fra keyboard
  Ikke-interaktivt: input fra fil

Features
- Synkron el asynkron kommandoutførelse
- Redirigering av stdin, stdout og stderr
- Kontroll over en kommando's environment
- Et antall built-in kommandoer (builtins)

Interaktive features
- Aliaser
- Kommandohistorikk
- Kommandolinjeredigering
- Jobbkontroll
  ...

PL features
- Variabler
- Flytkontroll-konstruksjoner
- Quoting
- Funksjoner
  ...
