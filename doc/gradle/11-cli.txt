CLI

gradle
  $ gradle                            help
  $ gradle --help                     cmdline opts
  $ gradle tasks --all                list all tasks
  $ gradle help --task <task>         task help

  $ gradle -q <task> <task> ...       run primary task(s) quiet
  $ gradle -w <task> <task> ...       run primary task(s) warning
  $ gradle -i <task> <task> ...       run primary task(s) info
  $ gradle -d <task> <task> ...       run primary task(s) debug
