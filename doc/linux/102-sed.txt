SED

Usage
  sed [opts] '[addr]sedcmd[opts]' [files]

Options
  -E 
  -e sedcmd
  -f sedscript
  -i[suffix]
  -n 

Address
  N             line N
  $             last line
  /regexp/      match        
  /regexp/!     not match
  addr₁,addr₂   range
  addr₁,+N      addr₁ + N lines

Sedcmds 
  a text        append after line
  c text        change (replace) line
  d             delete line
  i text        insert before line
  s/r/t/f       substitute regexp r, with text t, using flags f
