  Enkel kommando
    cmd [opts] [args]
    Funksjon, builtin, extcmd, script

  Pipeline
    [!] cmd₁ [| or |& cmd₂] ...

  Liste
    pipeline₁ [; or & or && or || pipeline₂] ... [; or & or \n]

  Sammensatte kommandoer
    {}
      { list; }
      Bash bruker {} for gruppering
      Bash kjører list i the current shell context
    
    ()
      ( list )
      Bash bruker () for gruppering
      Bash oppretter et subshell [execution environment]
      Bash kjører list i subskallet
    
    Boolsk expr
      [[ expr ]]

    Aritmetisk expr
      (( expr ))
    
    if
      if cmds; then cmds; [elif cmds; then cmds;] ... [else cmd;] fi
    
    case
      case word in [pattern [| pattern]...) cmds ;;]... esac

    select
      select name [in words ... ;] do cmds; done
    
    for
      for name [in words ... ] ; do cmds; done
      May use break or continue 

    for
      for (( expr₁ ; expr₂ ; expr₃  )) ; do cmds ; done
      May use break or continue 
     
    until
      until cmds₁; do cmds₂; done
      May use break or continue 
    
    while
      while cmds₁; do cmds₁; done
      May use break or continue 
    
  Coprosess
    coproc [name] cmd [redir]
