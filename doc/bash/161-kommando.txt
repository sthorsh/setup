GRAMMATIKK

Kommando
  cmd [args] [redir]
  name=val ... cmd [args] [redir]
   
Pipeline
  [!] cmd [| cmd]... (duplikat kjøremiljø)
  [!] cmd [|& cmd]...
   
Liste            
  pipeline [; pipeline]...  [; or & or \n]
  pipeline [& pipeline]...  [; or & or \n] (duplikat kjøremiljø)
  pipeline [&& pipeline]... [; or & or \n]
  pipeline [|| pipeline]... [; or & or \n]
   
Sammensatte kommandoer
  Uttrykk
    ((arithmetic-expr))
    [[ predicate-expr ]]                                   
    
  if               
    if cmds; then cmds; ... fi
    
  case             
    case word in pattern) cmds ;;... esac
    
  select           
    select name in words; do cmds; done
    
  for              
    for name in words; do cmds; done
    for (( expr; expr; expr )); do cmds; done
    
  while            
    while cmds; do cmds; done
    
  until            
    until cmds; do cmds; done
    
  Gruppe
    ( list ) (duplikat kjøremiljø)                                     
    { list; }
    
Coprosess        
  coproc [name] cmd [redir] (duplikat kjøremiljø)
