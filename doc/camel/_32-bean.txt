DSL
  .bean(beanType, method)
  .bean(beanName, method)
  .bean(beanReference, method) 
  .to("bean:beanType|beanName?method=name")

Bean
  beanType: vha reflection
  beanName: vha registeret
  beanReference: vha DI el new

Velge metode
  Alltid oppgi metodenavn
  Bruk @Handler ved overloading

Binde params
  Param 1 til @Body
  Param N til CamelContext (built-in types)
  Param N til Exchange
  Param N til Message
  Param N til Exception
  Param N til Registry
  Param N til TypeConverter
  Param N til @Header (annotations)
  Param N til @Headers
  Param N til @Attachments
  Param N til @ExchangeProperty 
  Param N til @ExchangeProperties
  Param N til @ExchangeException
  Param N til @Bean (language annotations)
  Param N til @Constant
  Param N til @JsonPath
  Param N til @Simple
  Param N til @XPath
  ...
  Evt bind i DSL'en dvs .bean(beanType, "foo(...)")
  Overflødige params gis tom verdi, overflødige args ignoreres
