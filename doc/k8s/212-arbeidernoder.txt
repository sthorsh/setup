ARBEIDERNODER

Kubelet
  K8s agent installert på en linux host
  Registrerer host som node i clusteret
  Poller apiserver på master for arbeid
  Administrerer application/system pods
  Rapporterer (feil) tilbake til master
  Eksponerer endepunkt <ip>:10255/pods
  Eksponerer endepunkt <ip>:10255/health
  Eksponerer endepunkt <ip>:10255/spec
  ...
  
Container engine
  Cf docker engine
    
Kube proxy
  K8s networking
  Gir hver pod en ip-addr
  Gir hver container i pod en tcp port
  Sørger for lastabalansering for pods → service → pods
