Image register
  search term                     docker.io
  login register                  docker.io, ...
  logout register                 https://index.docker.io/v1/, ...
  pull register/repo:tag          pull manifest
                                  pull compressed layers (distribution hash)
                                  uncompress layers (content hash)
                                  store layers
  push register/repo:tag          push manifest
                                  compress layers
                                  push compressed layers (distribution hash)

Image → container
  create img                      ny container

  run img [cmd]                   search; pull; create; start
  run -d ...                      detach
  run -i ...                      interactive
  run -p p₀:p₁ ...                host port p₀ → container port p₁
  run -P ...                      random host ports → exposed container ports
  run -t ...                      tty
  run -v dir₀:dir₁ ...            volume
  run --rm ...                    remove
