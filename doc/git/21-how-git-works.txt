Git er et VCS
  Distribuert VCS
  VCS
  Content tracker
  Persistent map: ~/,git/objects

Git er et filsystem
  Fil: blob
  Katalog: tree
  Versjon: commit

Objektmodell (graf)
  Commit → commit, tree
  Tree → tree, blob
  Blob 
  Tag → commit

Branch
  En ref til en commit: .git/refs/heads/...
  Default branch: .git/refs/heads/master

HEAD
  En ref til current branch: .git/HEAD 
  Eller en ref til en commit (detached HEAD)

Tag 
  En ref til en commit: .git/refs/tags/
  En tag er som en branch som aldri flytter seg

Merge
  Merge commit: en commit med to parent commits
  Fast-forward merge: move current branch tip to target branch tip
  Merges bevarer historien (graf)

Rebasing
  Reapply commits on top of another base tip (change base)
  Rebasing bevarer ikke historien (fra graf til linje)
