Simple command
                Splitter cmdline i cmds
                Setter opp I/O for cmdline
                For hver cmd do...
      Tokeniserer vha metategn
    History expansion !
    Alias expansion
                Opening keyword if mfl
    Brace expansion {}
    Tilde expansion ~
    Param expansion $ (svak quoting)
    Aritmetisk expansion $(()) (svak quoting)
    Command substitusjon $() (svak quoting)
        Tokeniserer vha $IFS
    Pathname expansion * ?
    Command lookup
    Setter opp I/O for cmd mm
    Utfører cmd
    Cmd returnerer exit code 0..255
  Done

Metategn              
  ␣ ⭾ < > | ; & ␤ ( )

  pre{str,...}post

  ~  = $HOME
  ~+ = $PWD
  ~- = $OLDPWD
