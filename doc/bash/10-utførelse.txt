  Ekspander history event ! ^
  Tokeniser vha metategn
  Klassifiser tokens
  Ekspander alias
  Parse tokens i enkle og sammensatte cmds
  Ekspander brace {n|char..n|char..n}...
  Ekspander tilde ~ ~+ ~- ~user
  Ekspander param ${}
  Kommandosubstitusjon $()
  Ekspander aritmetikk $(())
  Prosesssubstitusjon ()
  Splitt ord vha $IFS
  Ekspander wildcards
  Fjern quotes \ "" ''
  Rediriger stdin, stdout, stderr
  Ekspander og tilordne rhs i var=rhs [cmd] ...

Utførelse
  Søk-utfør funksjon
  Søk-utfør builtin
  Søk-utfør exec fil i hash
  Søk-utfør exec fil i PATH
  Hvis cmd med /: utfør i execution environment
  Hvis søk → hit: utfør i execution environment
  Vent på fullførelse
  Lagre exit code i $?
