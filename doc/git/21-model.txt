MODEL

Git
  Map (bytesekvens → hash)
  Persistent map (.git)
  Stupid content tracker (commit)
  High-level versioned file system
  Version control system (log, branch)
  Distributed version control system (remote)

---
Blob
  Et objekt
  Arbitrary content e.g tekstfil
  
Tree
  Et objekt
  En katalog
  Peker til fil(er)
  Peker til katalog(er)

Commit 
  Et objekt
  Peker til tree (snapshot)
  Peker til parent commit(s)
  Unreachable commit: kan ikke nås vha branch el tag

Annotert tag
  Et objekt
  Peker til objekt e.g commit
  RO dvs kan ikke flyttes (?)

Branch
  En peker
  b1 → commit
  RW dvs kan flyttes

HEAD
  En peker
  HEAD → branch → commit
  HEAD → commit (detached HEAD)

Tag
  En peker
  t1 → commit
  RO dvs kan ikke flyttes (?)

-----
.git/
  HEAD → branch → commit
  HEAD → commit
  objects/blob
  objects/tree
  objects/commit
  objects/annotert_tag
  refs/heads/branch → commit (rw)
  refs/tags/tag → commit (ro)
